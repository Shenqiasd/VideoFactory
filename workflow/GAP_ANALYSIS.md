## 🔧 已识别的技术债

### 1. 安全风险 - 优先级: 高
- ❌ **凭证明文存储**: `config/settings.yaml` 包含 LLM API key
- ❌ **硬编码 API key**: `scripts/groq_whisper_proxy.py`
- ❌ **无用户认证**: API 服务无鉴权
- ❌ **无权限管理**: 所有用户权限相同

**影响**: 生产环境无法部署

**建议**: 使用环境变量、引入 JWT 认证、实现 RBAC

**预估工作量**: 1周

### 2. 数据一致性风险 - 优先级: 中
- ❌ **时间字段不一致**: `api/routes/pages.py` 解析时间戳格式不一致
- ❌ **target_platforms 缺失**: 任务模型未持久化目标平台字段
- ❌ **并发写入无锁**: 多进程同时修改任务可能冲突

**影响**: 页面过滤异常、数据丢失风险

**建议**: 统一时间戳格式、完善任务模型、引入文件锁

**预估工作量**: 3天

### 3. 可移植性风险 - 优先级: 中
- ❌ **路径硬编码**: `scripts/start_all.sh` 中 KlicStudio 路径、FFmpeg 路径
- ❌ **依赖环境耦合**: 依赖本机 Python 3.11、特定版本 FFmpeg

**影响**: 无法在其他机器部署

**建议**: 配置文件覆盖、Docker 容器化

**预估工作量**: 2天

### 4. 监控和可观测性 - 优先级: 中
- ❌ **无性能监控**: 无法追踪任务处理时长、成功率
- ❌ **无告警机制**: 任务失败仅记录日志,无主动告警
- ❌ **无链路追踪**: 无法排查跨模块调用问题

**影响**: 故障发现滞后、排障困难

**建议**: 引入 Prometheus + Grafana、集成 Sentry

**预估工作量**: 1周
